3. REST nima?

REST — API yozishning qoidalari.

Eng ko'p ishlatiladigan metodlar:

Method	Vazifasi
GET	Ma'lumot olish
POST	Yangi ma'lumot yaratish
PUT	To'liq yangilash
PATCH	Qisman yangilash
DELETE	O'chirish

Misol:

GET /students

→ barcha talabalar

GET /students/5

→ 5-ID dagi talaba

POST /students

→ yangi talaba

DELETE /students/5

→ 5-ID dagi talabani o'chirish

4. FastAPI nima?

FastAPI — Python'da REST API yozish uchun zamonaviy framework.

Nega uni tanlaymiz?

✅ Juda tez

✅ Kod kam yoziladi

✅ Type hints ishlatadi

✅ Swagger hujjatlarini avtomatik yaratadi

✅ Backend bozorida talab yuqori
